Description

This is a basic calculator that performs simple statistical analysis on a small set of numbers with the following outcomes:

  • Maximum
  • Minimum
  • Average
  • Median
  • Range

Instructions

Choose three numbers to place in the calculator boxes and then press the "calculate" button to find their maximum, minimum, average, median, and range values.
